# LUT Asset Library

This folder keeps source metadata for downloadable LUT packs.

Downloaded third-party archives and extracted LUT files are intentionally ignored by Git:

- `assets/luts/raw/`
- `assets/luts/extracted/`

Use `catalog.json` as the tracked source-of-truth for provider URLs, local cache paths, and licensing notes. Do not redistribute third-party LUT packages from this repository unless their license has been explicitly cleared.
